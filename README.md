# Habit-tracker

Веб-приложение для управления привычками, построенное на Django REST Framework.

## Описание проекта

Habit-tracker — это API для создания, отслеживания и управления привычками. Проект позволяет пользователям создавать привычки, устанавливать напоминания, связывать привычки между собой и делиться публичными привычками.

## Основные возможности

- Создание и управление личными привычками
- Настройка времени, места и периодичности выполнения привычки
- Поддержка приятных привычек (без вознаграждения)
- Возможность связывать привычки между собой
- Система вознаграждений за выполнение привычек
- Публикация привычек для других пользователей
- Валидация данных на уровне модели и сериализатора

## Архитектура

Проект построен по классической архитектуре Django:

- **Модели**: `Habit` с полями для управления привычками
- **Сериализаторы**: `HabitSerializer` для преобразования данных
- **Представления**: Классовые представления DRF для обработки запросов
- **URL-ы**: Маршрутизация через Django URL Dispatcher

## API Эндпоинты

- `GET/POST /api/habits/` - список и создание привычек (авторизованные пользователи)
- `GET/PUT/PATCH/DELETE /api/habits/<id>/` - управление конкретной привычкой
- `GET /api/habits/public/` - список публичных привычек (все пользователи)

## Правила валидации

- Время выполнения привычки не должно превышать 120 секунд
- Периодичность не должна быть больше 7 дней (нельзя выполнять реже, чем раз в 7 дней)
- Приятная привычка не может иметь вознаграждения или связанной привычки
- Связанная привычка должна быть приятной
- Нельзя одновременно указывать вознаграждение и связанную привычку

## Технологии

- Python 3.x
- Django 4.x
- Django REST Framework
- PostgreSQL (предположительно)

## Установка и запуск

1. Клонируйте репозиторий
2. Установите Poetry: `pip install poetry`
3. Установите зависимости: `poetry install`
4. Активируйте виртуальное окружение: `poetry shell`
5. Выполните миграции: `python manage.py migrate`
6. Запустите сервер: `python manage.py runserver`
